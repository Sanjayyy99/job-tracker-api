# Job Application Tracker API

A backend REST API to manage and track job applications.

## Features
- CRUD operations
- Status tracking (Applied, Interview, Offer, Rejected)
- Filter by status
- Clean layered architecture

## Tech Stack
- Java
- Spring Boot
- MySQL

## Endpoints
POST /applications  
GET /applications  
PUT /applications/{id}  
DELETE /applications/{id}

## Future Improvements
- JWT Authentication
- Swagger UI
